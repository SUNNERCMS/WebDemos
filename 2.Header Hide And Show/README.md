## 版本功能说明
#### version1-2
完成了头尾栏滚动隐藏和显示的功能  
#### version3
（1）用nodejs搭建了http-server，并建立了mock模拟数据，以及路由设置。   
（2）使用Jquery进行了Ajax请求，由http-server处理后返回JSON数据。  
（3）使用handlebars模板引擎对请求响应的JSON数据进行了HTML渲染。
